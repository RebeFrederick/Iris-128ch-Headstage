The table below shows the specifications for the Iris-128S PCBs.

| :---------------------------: | :------------: | :-------------: |
| Min. Trace Width/Space        | 3 mil / 3 mil  | 5 mil / 5 mil   |
| # Layers                      | 8              | 4               |
| $\mu$Via (Hole/Diameter)      | 6 mil / 12 mil | None            |
| Thru-Hole Via (Hole/Diameter) | 8 mil / 14 mil | 10 mil / 20 mil |
| Board Thickness               | 1 mm           | 1.6 mm          |
| Surface Finish                | ENIG           | ENIG            |
| Copper Weight                 | 1 oz           | 1 oz            |
| Dielectric                    | Standard FR4   | Standard FR4    |
